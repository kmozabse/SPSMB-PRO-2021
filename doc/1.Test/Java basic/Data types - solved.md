## Primitivní datový typy
### Celá čísla
| Keyword | Velikost |
|:-------:|:--------:|
|   byte  |     1    |
|  short  |     2    |
|   int   |     4    |
|   long  |     8    |

### Desetinná čísla

| Keyword | Velikost |
|:-------:|:--------:|
|   float  |     4   |
|  double  |     8   |

### Další
| Keyword | Velikost |
|:-------:|:--------:|
|   boolean  |     1   |
|  char  |       2   |

Velikosti jsou uvedny v Bytes.

long myLong = 5L;
